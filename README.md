## Development

You can develop this project.

### Environment

- [Xcode](https://apps.apple.com/jp/app/xcode/id497799835): 13.3

### Configuration

- UI implementation: Storyboard + XIB
- Architecture: VIPER
- UITesting architecture: Page Object Pattern
- Branching model: Git-flow

### Setup

1. Clone the project.

    ```shell
    $ git clone https://github.com/uhooi/UhooiPicBook.git
    $ cd UhooiPicBook
    ```

2. Enable faster builds for Swift projects. (Optional)

    ```shell
    $ defaults write com.apple.dt.XCBuild EnableSwiftBuildSystemIntegration 1
    ```

3. Run `make setup` .  
After setup is complete, Workspace automatically opens in Xcode.
